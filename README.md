# 🎬 MovieVerse DevOps Project

A complete DevOps project demonstrating CI/CD, Infrastructure as Code, Docker, Kubernetes, and AWS.

## 🚀 Technologies Used

- AWS EC2
- Terraform
- Jenkins
- Docker
- Docker Hub
- Kubernetes (Minikube)
- Nginx
- HTML
- CSS
- JavaScript
- Git & GitHub

## 📂 Project Structure

```
MovieVerse-Devops/
├── website/
├── Terraform/
├── Kubernetes/
├── Dockerfile
├── Jenkinsfile
└── README.md
```

## ⚙️ Features

- Infrastructure provisioned using Terraform
- Jenkins CI/CD pipeline
- Docker image build and push to Docker Hub
- Kubernetes deployment using Deployment and Service manifests
- Movie data fetched using TMDb API

## 🔄 CI/CD Workflow

GitHub → Jenkins → Docker Build → Docker Hub → Kubernetes Deployment
